# Sweet Rosé Dream Ultimate
*The peak evolution of cutecore minimalism.*

**Sweet Rosé Dream (v9.3.8)** is more than just a fresh coat of paint. It is a complete structural overhaul of the Discord interface (optimized for Vencord, BetterDiscord, and Vesktop) built to eliminate visual clutter and bring a sense of calm to your daily chats.

---

## ⚠️ Important Compatibility Note
**Sweet Rosé Dream is a project developed and optimized exclusively for Discord’s Light Mode.** 
While a functional dark mode is included, it is provided strictly as a secondary aesthetic variant. This theme's core architecture, color depth, and glassmorphism metrics are engineered to perform best in the high-key, pastel-centric environment of the Light Mode interface.

---

## 🎀 Core Features
### Aesthetic Foundation
* **Exclusive Palette:** Crisp whites and ultra-soft gray backgrounds (`#fdfafd`) paired with balanced pastel pink accents (`#ed8695`).
* **Charming Typography:** Native integration of Google Fonts' **'Fuzzy Bubbles'** for a cozy, handwritten look.
* **Smart Readability:** WCAG AA Compliance ensures that usernames and nicknames remain perfectly legible in `gg sans`, regardless of the theme's vibe.

### Visual Effects & Glassmorphism
* **Translucent Profiles:** User modals, popouts, and profile cards feature crystal-like backgrounds with balanced `backdrop-filter: blur(12px)`.
* **Depth System:** Custom shadows (`var(--shadow-low, medium, high)`) create a floating panel effect.
* **Cutecore Scrollbars:** Smooth pastel pink gradients with fully rounded tracks.

### Functional Optimization (Anti-Bloatware)
We have surgically removed distracting commercial elements:
* Hidden Shop, Nitro Quests, and Nitro Home.
* Hidden Gift Nitro buttons and App Launcher shortcuts.

## 🚀 Installation Guide
This theme is compatible with modified Discord clients (Vencord, Vesktop, BetterDiscord).

### Option 1: URL Installation (Recommended)
1. Open Discord **User Settings**.
2. Go to the **Vencord** (or Themes) section.
3. Paste the RAW link below into the **Theme URL** field and press Enter:
   `https://raw.githubusercontent.com/Sk3tch1/Sweet-Rose-Dream-Discord/refs/heads/main/sweet-rose-dream.theme.css`

### Option 2: Manual Installation
1. Download the `sweet-rose-dream.theme.css` file from this repository.
2. Open your Discord Themes folder via your client's settings.
3. Move the file into the folder and toggle it on in the client.
