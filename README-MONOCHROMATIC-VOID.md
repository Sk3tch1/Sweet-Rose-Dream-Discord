# Monochromatic Void - Discord Theme
**Monochromatic Void (v9.3.7)** is a brutalist, high-contrast, and edgy monochrome interface for Discord. Engineered for users who demand zero visual distractions, it strips away the noise to provide a pure, immersive black, white, and gray experience.

## Introduction
Monochromatic Void is a structural overhaul designed for performance and aesthetic sharpness. It utilizes a brutalist design language, featuring precise border radii, deep "true black" backgrounds, and the technical aesthetic of the *Space Mono* typeface.

This theme is built to eliminate UI fatigue. By focusing on pure grayscale values and high-contrast typography, it provides a clean, professional, and aggressive workspace.

Development Note: 100% of the CSS architecture, structural debugging, and performance optimizations were generated and polished in collaboration with AI engineering models (Gemini/DeepSeek) to ensure a lightweight and bug-free experience.

## Main Features
**Aesthetic Foundation**
* **Brutalist Palette:** A strict grayscale hierarchy, utilizing true black (`#000000`) backgrounds and crisp white (`#ffffff`) accents.
* **Technical Typography:** Full integration of *Space Mono*, providing a clean, terminal-like feel that enhances readability and adds an "edgy" character to your chats.
* **WCAG AAA Compliance:** High-contrast text fields and links ensure maximum legibility for all users.

**Visual Effects and Glassmorphism**
* **Darkcore Aesthetics:** Features sharp, brutalist glassmorphism with heavy backdrop blurs, creating a depth-focused UI that feels solid and responsive.
* **Sharp Geometry:** Standardized 4px-8px border radii for a rigid, geometric look that breaks away from the "rounded" style of default Discord.
* **Dynamic Shadows:** An intense, layered elevation system (`var(--shadow-low, medium, high)`) that makes panels appear to float above the "void" background.

**Functional Optimization (Anti-Bloatware)**
The client interface has been cleaned up at the code level to remove commercial and distracting elements:

* Hidden the Shop button, Nitro Quests, and Nitro Home tab.
* Hidden the Gift Nitro button and App Launcher shortcut.

**Pro Modules and Extras**
* **"Darkcore" Splash Screen:** Replaces the standard loading screen with an animated, shimmering "DARKCORE" text element.
* **Grayscale Status Indicators:** All status colors are mapped to a clean, desaturated grayscale spectrum.

## Installation Guide

This theme is compatible with modified Discord clients that support custom CSS injection (Vencord, Vesktop, or BetterDiscord).

**Option 1: URL Installation (Recommended)**
1. Open Discord User Settings > Vencord/Themes.
2. Paste the RAW link into the theme URL field:
`https://raw.githubusercontent.com/Sk3tch1/Sweet-Rose-Dream-Discord/refs/heads/main/monochromatic-void.theme.css`
3. Press Enter to apply.

**Option 2: Manual Installation (BetterDiscord)**
1. Download the `monochromatic-void.theme.css` file from this repository.
2. Place the file in your BetterDiscord themes folder.
3. Enable the theme in Discord Settings.
